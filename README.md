# cookinghub-api

REST API for CookingHub - a recipes sharing application

## Prerequisites

```console
sudo apt install nodejs

sudo apt install git
```

## Setup

```console
git clone https://github.com/GBedenko/cookinghub-api

cd cookinghub-api

npm install
```

## Start Server


```console
npm start
```


## Testing

```console
npm test
```
