cookinghub-api

REST API for CookingHub - a recipes sharing application

Prerequisites

sudo apt install nodejs

sudo apt install git

Setup

git clone https://github.com/GBedenko/cookinghub-api

cd cookinghub-api

npm install

Start Server

npm start

Testing

npm test
